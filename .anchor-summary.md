## Goal
- Build a desktop app suite (EduSuite) for educational institutions that replaces manual Excel workflows with SQLite-backed CRUD modules for grades, students, teachers, enrollment/payments, and includes a spreadsheet editor with bidirectional DB sync.

## Constraints & Preferences
- Fedora 44 Workstation, Wayland, GNOME
- Python + PyQt5 (via PySide6 compat layer at `~/EduSuite/PySide6/__init__.py`)
- SQLite puro (sin ORM), ANSI-compatible SQL
- HiDPI scaling enabled: `QT_AUTO_SCREEN_SCALE_FACTOR=1`, `AA_EnableHighDpiScaling`
- App installed at `~/EduSuite/`, launchable via `.desktop` menu entry
- Español + XAF currency
- Escala de notas 0–10, 3 trimestres (T1/T2/T3)
- Interfaz profesional tipo Open Design, colores oscuros refinados (#0f1117 fondo, #e8e8ed texto)
- **Desktop app, no web app** — el editor con navegador es workaround, no el objetivo
- **No empaquetar** hasta orden explícita
- **No cambiar** lo establecido sin instrucción directa

## Key Decisions
- SQLite puro sin ORM para mantener migración futura a PostgreSQL sin cambiar queries
- Sync offline vía Excel (export/import con diff visual) como mecanismo multi-usuario
- Matrícula implementada como control de pagos por año escolar con monto fijo configurable por año
- Escala de notas 0–10, 3 trimestres fijos (T1/T2/T3), formato Excel con 1 hoja por grupo
- Editor con servidor HTTP local + navegador del sistema (workaround hasta que se integre QWebEngineView)
- Diseño profesional "Open Design": paleta oscura refinada, bordes sutiles (8–14px radius), iconos Unicode para navegación, acento con barra de color en cards y sidebar, inputs y combos con focus ring
